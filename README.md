# 🏦 Bank Management System

A desktop-based Bank Management System developed using **Java Swing (JFrame)** and **MySQL**.  
This project simulates ATM functionalities with secure login and banking operations.

## 🚀 Features
- User Registration & Login
- Account Number & PIN Generation
- Deposit Money
- Withdraw Money
- Fast Cash
- Balance Enquiry
- Mini Statement
- PIN Change
- Account Details

## 🛠️ Technologies Used
- Java
- Java Swing
- JDBC
- MySQL

## ▶️ How to Run
1. Clone the repository
```bash
git clone https://github.com/your-username/Bank-Management-System.git
```

2. Configure MySQL database in `Conn.java`

3. Run:
```bash
Login.java
```
